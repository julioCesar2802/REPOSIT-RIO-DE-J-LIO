PYTHON

  1. QUAL A HISTÓRIA DO PYTHON?

    • Python foi criada pelo programador holandês Guido van Rossum, que começou a desenvolvê-la em 1989 como um projeto de passatempo para o Natal, inspirado na série de TV "Monty Python's Flying Circus". A linguagem foi lançada publicamente em 1991, com a versão 0.9.0, e evoluiu através de importantes lançamentos, como Python 2.0 (2000) e Python 3.0 (2008), que se tornou a versão atual e a mais popular devido à sua natureza de código aberto e à forte comunidade de desenvolvedores.

  • QUEM CRIOU O PYTHON?

    • O criador do Python é o programador holandês Guido van Rossum. Ele começou a desenvolver a linguagem em 1989, como um projeto de hobby, e a lançou oficialmente em 1991. O nome foi escolhido em homenagem ao seriado britânico de comédia "Monty Python's Flying Circus".

  • POR QUE OS PROGRAMADORES INICIANTES ESCOLHEM PYTHON COMO PRIMEIRA LINGUAGEM?

    • Programadores iniciantes escolhem Python como primeira linguagem principalmente devido à sua sintaxe simples e legível, que se assemelha à linguagem humana, facilitando a compreensão dos conceitos fundamentais de programação.
